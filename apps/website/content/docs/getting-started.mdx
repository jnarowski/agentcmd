# Getting Started

This guide will help you get AgentCmd up and running on your local machine.

## Prerequisites

Before you begin, ensure you have:

- Node.js 18+ installed
- pnpm package manager
- Git

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/sourceborn/agentcmd.git
   cd agentcmd
   ```

2. **Install dependencies**

   ```bash
   pnpm install
   ```

3. **Build the project**

   ```bash
   pnpm build
   ```

## Development

Start the development server:

```bash
pnpm dev
```

This will start:
- Frontend dev server on `http://localhost:5173`
- Backend API on `http://localhost:3456`
- Inngest UI on `http://localhost:8288`
