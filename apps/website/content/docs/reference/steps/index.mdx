# Step Types Reference

Complete API documentation for all 8 workflow step types.

## Quick Reference

| Step | Purpose | Timeout | Use For |
|------|---------|---------|---------|
| [agent](/docs/reference/steps/agent) | Run AI CLI tools (Claude, Codex, Gemini) | 30 min | Interactive AI with file editing |
| [ai](/docs/reference/steps/ai) | Non-interactive AI generation | 5 min | Quick AI responses, structured data |
| [cli](/docs/reference/steps/cli) | Execute shell commands | 5 min | Build, test, deploy scripts |
| [git](/docs/reference/steps/git) | Git operations | 2 min | Commit, branch, PR creation |
| [artifact](/docs/reference/steps/artifact) | Upload files/directories | 5 min | Save outputs for review |
| [annotation](/docs/reference/steps/annotation) | Progress notes | Instant | Timeline milestones |
| [phase](/docs/reference/steps/phase) | Group related steps | N/A | Organize workflow stages |
| [log](/docs/reference/steps/log) | Logging | Instant | Debug, track execution |

## Step Categories

### AI/Agent Steps

Run AI models:
- **agent** - Full CLI tool execution (Claude Code, Codex, Gemini)
- **ai** - Quick AI generation (text or structured output)

### Execution Steps

Run commands and operations:
- **cli** - Shell command execution
- **git** - Version control operations

### Output Steps

Save and display results:
- **artifact** - Upload files for download
- **annotation** - Timeline progress notes
- **log** - Console logging

### Organization Steps

Structure workflows:
- **phase** - Group steps into logical stages

## Common Patterns

### AI â†’ Git Flow

```typescript
// Generate code
const result = await step.agent("implement", {
  agent: "claude",
  prompt: "Implement user authentication",
});

// Commit changes
await step.git("save", {
  operation: "commit",
  message: "feat: Add authentication",
});
```

### CLI Chain

```typescript
await step.cli("install", { command: "pnpm install" });
await step.cli("build", { command: "pnpm build" });
await step.cli("test", { command: "pnpm test" });
```

### Parallel Execution

```typescript
await Promise.all([
  step.cli("lint", { command: "pnpm lint" }),
  step.cli("typecheck", { command: "pnpm check-types" }),
  step.cli("test", { command: "pnpm test" }),
]);
```

### Progress Tracking

```typescript
await step.annotation("start", { message: "Starting deployment..." });
await step.cli("deploy", { command: "deploy-prod" });
await step.annotation("complete", { message: "Deployment successful!" });
```
