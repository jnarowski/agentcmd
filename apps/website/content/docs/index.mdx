# Welcome to AgentCmd

AgentCmd is a powerful AI-driven CLI workflow automation tool that helps you orchestrate complex tasks with ease.

## Features

- **AI-Powered Workflows**: Leverage AI agents to automate complex CLI tasks
- **Workflow Engine**: Build and manage sophisticated workflow pipelines
- **Real-time Updates**: WebSocket-based real-time status updates
- **Extensible**: Easy to extend with custom commands and integrations

## Getting Started

Ready to get started? Check out the [Getting Started](/docs/getting-started) guide.

## Quick Example

```bash
# Install AgentCmd
pnpm install

# Start the application
pnpm dev

# Access the UI
open http://localhost:5173
```
