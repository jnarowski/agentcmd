Build multi-step AI workflows with <<AgentCMD /> /> that orchestrate Claude, OpenAI Codex, and Google Gemini to automate complex development tasks. Define workflows once, run them anywhere.

## What is <<AgentCMD /> />?

Think of it as **GitHub Actions for AI agents**. You define workflows with phases and steps, then <<AgentCMD /> /> handles execution, monitoring, and artifact management.

```typescript
// Simple 3-step workflow
defineWorkflow(
  {
    id: "deploy-feature",
    phases: ["plan", "implement", "deploy"],
  },
  async ({ step }) => {
    await step.phase("plan", async () => {
      await step.agent("architect", {
        agent: "claude",
        prompt: "Design the authentication system",
      });
    });

    await step.phase("implement", async () => {
      await step.agent("code", {
        agent: "codex",
        prompt: "Implement the design from the planning phase",
      });
    });

    await step.phase("deploy", async () => {
      await step.git("push", {
        operation: "commit-and-branch",
        message: "feat: Add authentication system",
      });
    });
  }
);
```

## Quick Example

Here's a complete workflow that generates a feature spec, implements it, and creates a PR:

```typescript
import { defineWorkflow, buildSlashCommand } from "agentcmd-workflows";

export default defineWorkflow(
  {
    id: "ship-feature",
    phases: ["spec", "implement", "review", "ship"],
  },
  async ({ event, step }) => {
    const { featureName } = event.data.args;

    // Generate spec
    await step.phase("spec", async () => {
      await step.agent("generate-spec", {
        agent: "claude",
        json: true,
        prompt: buildSlashCommand("/cmd:generate-feature-spec", {
          context: featureName,
        }),
      });
    });

    // Implement
    await step.phase("implement", async () => {
      await step.agent("implement", {
        agent: "codex",
        prompt: buildSlashCommand("/cmd:implement-spec", {
          specIdOrNameOrPath: featureName,
        }),
      });
    });

    // Create PR
    await step.phase("ship", async () => {
      await step.git("create-pr", {
        operation: "pr",
        title: `feat: ${featureName}`,
        body: "Auto-generated by <AgentCMD /> workflow",
      });
    });
  }
);
```
