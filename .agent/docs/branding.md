# AgentCmd Branding Guide

## Project Identity

**AgentCmd** - Full-stack web application and CLI tool for orchestrating AI agent workflows with integrated chat, file editing, and terminal capabilities.

### Core Value Proposition
- Unified interface for multiple AI agents (Claude Code, OpenAI Codex, Google Gemini)
- Real-time collaboration with AI agents on coding tasks
- Session management, Git integration, and project workspaces
- Single-user personal development tool for individual developers

### Target Audience
- Individual developers and engineers
- AI/LLM enthusiasts
- CLI power users
- Open source developers

## Existing Design System

**UI Framework**: shadcn/ui (Radix UI + Tailwind CSS v4)

**Visual Style**:
- Modern, minimalist, tech-focused
- Dark mode by default
- High contrast, accessibility-focused
- Terminal/coding-centric aesthetic
- Developer-first, utilitarian design

**Typography**: Geist font family (Vercel's modern, clean font)

**Icons**: Lucide React (modern, clean icons)

**Technology Aesthetic**: DevTools-inspired UI, functional, no ornamentation

## Logo Design Prompts

### Prompt 1: Abstract Geometric

Create a minimalist logo for "AgentCmd" - a developer tool for AI agent orchestration.

**Icon Concept:**
- Abstract geometric mark suggesting flow/connection/intelligence
- Interlocking nodes or circuit-like pathways forming a distinctive shape
- Sharp angles, clean lines, precise geometry
- Suggests: network intelligence, workflow automation, command execution

**Color Palette:**
- Primary: Deep navy (#0A1628) or rich charcoal (#0F172A)
- Accent: Electric cyan (#06B6D4) or bright blue (#3B82F6)
- Optimized for dark backgrounds (#000000 to #0A0A0A)

**Wordmark:** "agentcmd" (all lowercase, technical/unix aesthetic)
- Modern geometric sans-serif (Geist, Inter, or similar)
- Tight kerning, clean spacing
- Optional: monospace influence

**Style:** Linear, Vercel, Stripe - ultra-minimal, high-tech, scalable

**Format:**
- Icon + wordmark version (horizontal lockup)
- Icon-only version (for app icons, favicons)
- Must work at 16px to large sizes
- Legible on dark (#0A0A0A) and light (#FFFFFF) backgrounds

## Design Recommendations

### Wordmark Style
**Recommended: "agentcmd" (all lowercase)**

Reasons:
- Unix/CLI convention (matches `npx agentcmd start`)
- Matches package naming (`@repo/agent-cli-sdk`)
- Developer audience expects lowercase commands
- Technical, approachable, not corporate

### Color Direction
**Cool blues/cyans with dark backgrounds**

- Primary: Deep slate or navy (#0F172A, #1E293B)
- Accent: Bright cyan (#06B6D4) - high contrast on dark, modern, tech-forward
- Works perfectly in dark mode UIs
- High accessibility contrast ratios

### Brand Attributes
- **Command-driven**: CLI heritage, terminal aesthetic
- **Intelligent/AI-focused**: Agent orchestration
- **Minimalist**: Clean tech aesthetic
- **Trustworthy**: Developer audience
- **Accessible**: High contrast, keyboard-friendly

### Design Inspiration
- Linear (minimalist geometric)
- Vercel (clean, sharp, modern)
- Anthropic Claude (trustworthy AI)
- GitHub (developer-focused)
- Warp terminal (modern CLI)

## Logo Usage Guidelines

### Icon Sizes
- **16px**: Favicon, browser tabs
- **32px**: App icons, taskbar
- **64px**: Social media avatars
- **128px+**: Marketing, website headers

### Color Variants
- **Primary**: Full color on dark background (default)
- **Monochrome**: Single color (white on dark, dark on light)
- **Inverted**: For light backgrounds

### Clear Space
Maintain minimum clear space around logo equal to height of "a" in wordmark

### Don'ts
- Don't add drop shadows or effects
- Don't outline the logo
- Don't rotate or skew
- Don't change proportions
- Don't use unapproved colors

## File Formats

### Production Files
- **SVG**: Primary format for web/digital
- **PNG**: Transparent backgrounds, multiple sizes
- **ICO**: Windows favicon
- **ICNS**: macOS app icon
